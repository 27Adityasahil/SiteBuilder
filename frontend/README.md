# SiteBuilder Frontend

Frontend application for SiteBuilder Visual Development Platform.

Built with React and TypeScript providing a powerful drag-and-drop website creation experience.

---

# Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS
- Redux Toolkit
- DND Kit
- React Router
- Axios


---

# Features


## Builder Interface

Three-panel editor layout:


LEFT PANEL

Canvas


RIGHT PANEL

Properties

Build:
